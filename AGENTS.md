#  Portafolio Alejandro Ajenjo
El proyecto es un portafolio que servira como un pagina para mostrar los conocimientos de la persona, y puestos de trabajo
## Stack
- Lenguaje:  HTML, SCSS, TypeScript estricto
- Framework React + Vite (SPA)
- Routing: React Router (URLs por idioma /es /ca /en)
- i18n: provider propio en src/i18n (API compatible con next-intl: useTranslations/useLocale)
- Tests:  Vitest
